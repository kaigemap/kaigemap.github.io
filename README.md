# kaigelab

kaigemap の個人制作物をまとめるリンク集兼ポートフォリオです。

GitHub Pages でそのまま公開できるように、ビルド不要の静的サイトとして構成しています。

## 更新方法

- `script.js` の `projects` 配列に制作物を追加・編集します。
- GitHub Pages のデモがある場合は `demoUrl`、コードだけの場合は `repoUrl` を設定します。
- スクリーンショットは `assets/screenshots/` に置き、`screenshotUrl` にパスを設定します。
- カテゴリは `tool`、`stream`、`design`、`experiment` のいずれかを使います。

## 公開

このリポジトリは `kaigemap.github.io` なので、GitHub Pages を有効にすると `https://kaigemap.github.io/` で公開できます。
